# Voice Controlled Home Automation Bot

## Objective
A virtual smart home automation system that controls lights, fans, and air conditioners using voice commands.

## Technologies Used
- Python
- SpeechRecognition
- PyAudio
- pyttsx3

## Features
- Light ON/OFF
- Fan ON/OFF
- AC ON/OFF
- Status Check
- Voice Feedback
- Continuous Listening

## Commands
- light on
- light off
- fan on
- fan off
- ac on
- ac off
- status
- exit
